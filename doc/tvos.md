架构：

1. 后台采用Nodejs+express来提供Restful API
2. 数据库采用 Mongodb，通过Mongoose驱动
3. 前端采用AngularJs进行开发
4. 前端JS代码采用模块化编写，使用Requirejs驱动
5. 前端UI使用了bootstrap。
6. 用grunt对项目进行构建工作
7. 代码托管用Git server（在实验室搭建）
